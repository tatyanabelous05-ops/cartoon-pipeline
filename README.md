# Cartoon Pipeline

Script analyzer for animation teams. Paste screenplay → get image & video prompts ready for Magnific Nodes.

## Deploy to Railway

1. Upload this folder to GitHub (new repository)
2. Go to railway.com → New Project → Deploy from GitHub
3. Select your repository
4. Go to Variables → Add: `GEMINI_API_KEY` = your key from aistudio.google.com
5. Deploy — Railway gives you a public URL to share with your team
